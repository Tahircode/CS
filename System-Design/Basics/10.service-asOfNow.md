
## our service design
- multiple servers
- load balancer
- master-slave DB
- cache DB
- CDN
- Stateless Web ( shared storage for state data)

![alt text](<0.public/image copy 15.png>)

lets analyise our service:
- user gets ip-address of load-balancer from DNS
- static content got fetched from CDN
- http req goes to load-balancer
- load-balancer distributes req evenly
- if data is cache -> cache return data
- otherwise data returns from slave-db
- if its session data : NoSQL data-base returns the data
