1. "Normal" Initialization with Parentheses ()
- This is the classic, traditional form of initialization in C++. It directly invokes the constructor that matches the arguments provided inside the parentheses.

- its more flexible: it allows us to perform omplex operations during initialization if needed, such as data validation or computation before assignment.

How it Works:
It's a straightforward call to a constructor.

int x(5);               // Directly initializes x to 5

string s("hello");      // Calls the string constructor that takes a const char*
vector<int> v(10, 0);   // Calls the vector constructor that takes a size and a value


// For a custom class
