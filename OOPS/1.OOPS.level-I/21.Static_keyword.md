# Static Keyword:
- The static keyword makes a variable or method belong to the class rather than the object. Static members are shared across all instances and can be accessed without creating an object. In Java, static blocks and static nested classes are also supported.

**1️⃣ static Variables**

*C++*

- a) Static data member (inside class)
class A {
public:
    static int x;
};

int A::x = 10;   // definition


Shared across all objects

Only one copy

Exists for entire program lifetime

- b) Static local variable (inside function)
void func() {
    static int count = 0;
    count++;
}

Initialized once

Retains value between function calls

*Java*

Static class variable
class A {
    static int x = 10;
}

One copy per class

Stored in method area / heap

Shared by all objects

**2️⃣ static Functions / Methods**
*C++*

class A {
public:
    static void show() {
        cout << "Hello";
    }
};


Can access only static members

Called using class name

A::show();

*Java*
class A {
    static void show() {
        System.out.println("Hello");
    }
}

Cannot use this or super

Cannot access non-static members directly

A.show();

**3️⃣ static Blocks (Java Only)**
class A {
    static {
        System.out.println("Static block");
    }
}


Runs once when class is loaded

Used for static initialization

❌ Not present in C++

**4️⃣ static Classes**
*Java*
class Outer {
    static class Inner {
        void show() {}
    }
}

Only nested classes can be static

No outer object required

*C++*

❌ No static class keyword
✔️ Similar behavior achieved via namespaces
