# copy assignment operator
- The copy assignment operator is a special function that defines what happens when you assign an already-existing object to another already-existing object using the = operator.
- If the copy constructor is like creating a new file by copying an existing one, the copy assignment operator is like overwriting the contents of an existing file with the contents of another. The destination object already exists; you're just changing its state.

- 
ClassName obj1 = className(values);
ClassName obj2 == obj1;

ClassName obj3 = className(values); //copy constructor

obj2 = obj3; //copy assignment opr
