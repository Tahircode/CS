# Contructor
  - this is an special function that is used to initialize member variables during onject creation

- A constructor is a special method within a class that is automatically called when an object of that class is created. Its primary purpose is to initialize the object into a valid, usable state.

- no return type
- same name as class name

- 1. default contructor: special contructor that is automatically generated by class if its not defined by the programmer.
it takes no parameters

- 2. Parameterized Constructor (takes parameters): special constr. that accepts one or more parameters during object creation

- Name: The name of the constructor must be exactly the same as the name of the class.
- Return Type: A constructor has no return type. Not even void. This is because its job is to create the object itself, not to return a value.
- Access Modifiers: Constructors are usually public(can make private->security) so that you can actually create objects from outside the class. However, they can be private or protected for specific design patterns (like Singleton).
- Automatic Invocation: You never call a constructor directly. It is invoked automatically when you create an object using the new keyword (for heap allocation) or by simply declaring the object (for stack allocation).

class Player {
private:
    std::string name;
    int health;
    int xp;

public:
    // 1. Default Constructor (takes no parameters)
    Player() {
        std::cout << "Default constructor called!" << std::endl;
        name = "None";
        health = 100;
        xp = 0;
    }

    // 2. Parameterized Constructor (takes parameters)
    Player(std::string name_val, int health_val, int xp_val) {
        std::cout << "Parameterized constructor called!" << std::endl;
        name = name_val;
        health = health_val;
        xp = xp_val;
    }
    
    // A simple method to display player info
    void display() {
        std::cout << "Name: " << name << ", Health: " << health << ", XP: " << xp << std::endl;
    }
};

int main() {
    // Creating an object using the Default Constructor
    Player empty_player; 
    empty_player.display(); // Output: Name: None, Health: 100, XP: 0

    std::cout << "---------------------" << std::endl;

    // Creating an object using the Parameterized Constructor
    Player hero("Aragorn", 150, 500);
    hero.display(); // Output: Name: Aragorn, Health: 150, XP: 500
    
    return 0;
}

class Person {

        this.age = age;
    }

    // Copy-like constructor
    Person(Person other) {
        this.age = other.age;
    }

3️⃣ Factory Method (Best Practice)
class Person {
    int age;

    private Person(int age) {
        this.age = age;
    }

    static Person copyOf(Person other) {
        return new Person(other.age);
    }
}

✔️ Clean
✔️ Explicit
✔️ Safe for deep copies
